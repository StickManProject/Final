Installer Stickman Galactica :

1. Dézipper les StickG.part.rar 

2. executer le fichier StickG.msi

3. Suivez les instructions de l'assistant d'installation*.

4. Une fois l'installation terminée le jeu se trouvera dans le dossier C:\Program Files (x86)\StickyG\StickG\

5. Bon jeu !

* accepter l'installation de Java Runtime Environment 8.
